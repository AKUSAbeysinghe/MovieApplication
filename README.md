# 🎬 Movie Explore App

A modern, responsive web application that allows users to discover, search, and explore a wide variety of movies. Built with [React](https://reactjs.org/) and powered by the [TMDB API](https://www.themoviedb.org/documentation/api), this app offers a clean and interactive UI for movie lovers.

## 🚀 Features

- 🔍 Search movies by title
- 🏆 View trending, top-rated, and upcoming movies
- 🎥 Detailed movie pages with poster, description, rating, genres, and release date
- 📱 Fully responsive design (mobile, tablet, desktop)
- 🌙 Light/Dark mode toggle
- ❤️ Add to Favorites (optional with localStorage or backend support)


## 🛠️ Tech Stack

- **Frontend**: React, TypeScript (or JavaScript), Tailwind CSS / CSS Modules / Styled Components
- **API**: TMDB (The Movie Database) API
- **Tools**: Axios, React Router, React Icons
